Why

This lets you begin moving into more complicated stuff.

If you're walking around and like the look of some flowers, you might want to take some home.

Well, the orange flowers are a little boring looking, while the red flowers are great.

So you'd use two branches to say: "When the flower is orange, ignore it. When the flower is red, pick it up!"

What

This can actually take two different forms, and really depends on what you want to do.

The first is still very basic, the "if-else".

All this does is say "If this statement is true, do this thing. If it isn't true, do this other thing." Simple.

The other, though, is where things get a little more complicated.

The "If-elseif", where you set up two different statements.

This says "If this equation is true, do thing one, if it isn't, check equation two. If equation two is true, do thing two. If neither are true, move on."

As you can see, it's a little more complicated, but still very understandable.


How

First, choose which type you want. "If-else", or "If-elseif".

Either way, you'll want to click on the little cogwheel in the top left of the "if" block.

Then, drag either the "else" or "else-if" blocks onto the main "if" block, just like other blocks.

Then, fill in the blocks with whatever things you want to run.
