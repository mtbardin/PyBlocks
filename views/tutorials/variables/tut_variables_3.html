Why

You'll use lists for storing larger amounts of information, like a list of names or ages.

It can be of any type, but I will warn you against putting different types of variables in a single list.

You can do it, but actually using that data for things other than printing will cause issues for you later.


What

Lists are just what the name implies. A list of variables.

The computer puts all these variables in a row so that you can check on them later.


How

Drag the one of the "create list" blocks onto the workspace, and fill it with what variables you want.

There's a cog in the top left that lets you add more spots for the variables if you need to.

You'll need to drag out and connect a variable block as well, so the computer knows how to get to the list later.
