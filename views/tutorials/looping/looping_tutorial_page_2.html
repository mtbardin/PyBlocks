Why

For loops give you greater control over the exact number of times the loop will run.
It's useful for if you want to, say, pick up a certain number of flowers.
This let's you do that easily, without putting down the same block 10 or 20 or however many times you would normally.

What

A for loop works by creating a variable to count the number of times it needs to repeat
It counts up from it's beginning to the end, repeating the blocks inside each time.
There are actually two different blocks, depending on what you want to do.
  • The first is a very basic one, that handles most everything on its own.
    It repeats the blocks inside of it a set number of times.

  • The second is a bit more in-depth, for people who want more control.
    It gives you access to the counting variable, and lets you set by how much it counts.
    So if you want to go from 1 to 10, but count by 3 each time, you can!

How

For the simple one, just put the number of times you want the loop to repeat.

For the more advanced one, you need three numbers.
The beginning number, the ending number, and the step number.
  1. The beginning number is the start, obviously.
  2. The end number means that when the counter is equal to, or bigger than it, the loops stop.
  3. And the step number is the amount that gets added to the counter each loop.

(It's called a step number because it's like when you run up stairs 2 or 3 steps at a time.)
