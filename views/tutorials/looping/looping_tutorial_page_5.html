Why

There will be times when you want to repeat an action within a loop.
This is called "nesting" a loop, since it's like a bird in a nest.
This is primarily used when you want to do something to each value in a list
(Or if you want to work with 2-Dimensional arrays, which we will teach you about later.)

What

There are two parts to a nested loop.
The 'outer' loop is the nest in the analogy, holding the 'inner' loop inside of it.

How

Drag the loop block you want to be the outer loop to the workspace.
Then, drag the loop block you want to be the inner loop to the workspace.
Finally, connect the inner block to the 'do' part of the outer loop and you are ready to go!
