Branching

What actually is this thing?

Branching is the most basic way to begin more complicated programming. For this tutorial, we'll focus on if-else conditions. There are three options for you.

"If" is followed by a condition, like the variable 'X' being greater than some number.
"Else" is not followed by a condition, because it holds everything that happens if the "if" statement is false.
"Else if" is a special option, in case you want to set up multiple checks using the same statement.

Why should you need to know this?

Normally, a computer will run all the commands you tell it to, one by one. Branching lets you tell it to run certain ones, depending on the condition you set up. It allows you to set up a system of checks that cause the program to "branch" out, similar to a tree.

It's one of the core parts of programming, and something you'll be using consistently.
