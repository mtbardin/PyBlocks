Why

The Grid World provides a useful way to actually see what your code is doing.
No more just looking at text on a screen, now you can see your wizard following the instructions you give it.
This lets you more easily approach and understand what's happening, and fix something if it goes wrong.

What

You'll have a little wizard that you instruct, and it does what you tell it to do.
He's searching the forest for some things to bring home, and you'll be helping him!
Some tutorials will have you pick flowers, others want you to look for treasure.
There are some snakes in the forest, but your wizard can scare them off with a quick spell.

How

Certain tutorials will let you help the wizard, but you can go to the "Interactive Tutorial" in the main menu too.
If you do, there's no set goal, but you can set up whatever code you want, and it will run.
If you want to, you can go try that now!
